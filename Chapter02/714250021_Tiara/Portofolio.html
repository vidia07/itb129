PORTOFOLIO TIARA HENDRI MAULIA

Deskripsi

Ini adalah tugas pemrograman tingkat I yang mana intruksinya buatlah 1 buat tampilan website Portofolio yang didalamnya berisi dengan file HTML dan CSS. Disini Saya menggunakan href untuk melanjutkan halaman selanjutnya di mana Saya membuat 2 web Deskripsi dan Profil. Saya juga menggunakan header dan container dimana saya terinspirasi dari tugas sebelumnya. Tugas ini bertujuan untuk memperdalam wawasan Saya dan membiasakan Saya untuk menggunakan HTML dan CSS.

Saya membuat tugas ini dengan sangat simpel, ada beberapa style yang terinspirasi dari tugas tugas sebelumnya. Saya sadar kekreatifan Saya kurang di tugas ini, tapi dengan begitu Saya akan berusaha untuk belajar kembali agar kecepatan Saya dalam menggunakan HTML dan CSS semakin meningkat hingga Saya dapat lebih cepat mengerjakan dan lebih banyak pula waktu Saya untuk memikirkan konsep tanpa mengejar deadline tugas.
