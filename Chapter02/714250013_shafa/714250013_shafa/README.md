Saya membuat tampilan website portofolio sederhana menggunakan HTML dan CSS. Website ini berisi informasi dasar tentang diri, seperti profil singkat, media sosial, dan motto.
Struktur terdiri dari:
1. index.html: halaman utama website yang menampilkan isi portofolio
2. style.css: file CSS untuk mengatur tampilan website agar lebih menarik
3. README.md: berisi deskripsi singkat mengenai tugas ini
