# Soal A — NPM GANJIL
## Landing Page “Future Tech App” (Modern & Interaktif)
1. Buat sebuah landing page futuristik untuk mempromosikan aplikasi teknologi masa depan (AI assistant, smart home, AR app, dll).
2. Syarat Utama:
- Gunakan HTML + CSS + JavaScript .
- Halaman wajib memiliki:
- Hero animatif (teks atau gambar muncul dengan efek glow/typing/slide-in).
- Section fitur dengan card interaktif (hover efek keren).
- Section video (embedded video atau animasi Lottie diperbolehkan).
- CTA (Call To Action) yang men-scroll otomatis ke bagian tertentu.
3. Desain high-tech neon, gradient, glassmorphism, atau dark mode.
4. Responsif (mobile–tablet–desktop).
5. Output:
- Lakukan Pull Request dan Push ke Repository Github ini
- Untuk Adlink Proyek lengkap + screenshot tampilan desktop & mobile masukan kedalam 1 file PDF dan kirimkan untuk dilakukan pengujian Coding di pertemuan berikutnya


# Soal B — NPM GENAP
## Landing Page “Creative Agency” (Premium & Profesional)
1. Buat landing page agensi kreatif dengan konsep clean, elegan, dan premium.
2. Syarat Utama:
- Gunakan HTML + CSS + JavaScript.
- Halaman wajib berisi:
- Hero section dengan parallax effect atau animasi smooth.
- Portfolio slider (gambar otomatis berganti atau swipe manual).
- Daftar layanan dengan icon modern.
- Testimoni dengan card yang dapat digeser.
- Form kontak dengan validasi JavaScript.
3. Desain elegan, minimalis, tipografi tegas, warna premium (white–gold, black–silver, dll).
4. Responsif (mobile–tablet–desktop).
5. Output:
- Lakukan Pull Request dan Push ke Repository Github ini
- Untuk Adlink Proyek lengkap + screenshot tampilan desktop & mobile masukan kedalam 1 file PDF dan kirimkan untuk dilakukan pengujian Coding di pertemuan berikutnya
